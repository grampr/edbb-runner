# EDBB Runner

EDBBのWebアプリから「起動」ボタンを押すことで簡単にDiscord Botを起動できます。

## ブラウザからBotを起動する

EDBBのWebアプリから「起動」ボタンを待ち受けます

1. `edbb-runnerを起動.bat`を起動する
2. コマンドプロンプト上にDiscord Botトークンを入力する
3. EDDBのWebアプリから「起動」ボタンを押す
4. Discord Botが起動します！
  (作成したBotのコードはbot.pyというファイルに保存されます)

## 既にbot.pyが存在する場合

Webアプリの待ち受けをせずに、前述の手順で作成したbot.pyを起動します

1. `bot.pyを起動.bat`を起動する
2. Discord Botが起動します！
